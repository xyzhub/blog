## 属性说明
| 属性           |      说明      |  类型 |  默认值 |
| ------------- | :-----------: | :----: |----: |
| type    | 设置按钮类型 | 'primary' \| 'default' \| 'dashed' \| 'success' \| 'warning' \| 'error' | primary |
| size    |   设置按钮大小    |   'sm' \| 'lg'   | 空 |
